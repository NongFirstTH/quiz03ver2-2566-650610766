# Quiz 03 : Public Chatroom API

หลังจากการ Fork และ Clone repository แล้ว, ให้เปิดโฟลเดอร์ด้วย VSCode และรันคำสั่งใน terminal:

```bash
npm install
npm run dev
```

ให้นักศึกษาสร้างระบบ API สำหรับห้องแชทสาธารณะโดยใช้ความรู้เรื่อง RESTful API บน Next.js และการยืนยันตัวตนด้วย JSON WEB TOKEN
